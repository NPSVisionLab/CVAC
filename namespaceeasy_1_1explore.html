Detailed Description

Easy Computer Vision

explore.py provides functionality to explore the capabilities of a
service.  It can determine:
* the type of algorithm,
* metadata about the algorithm,
* whether it can handle images and/or videos,
* whether it can process an assortment of file types,
* the validity of the result structure syntax,
and more.
