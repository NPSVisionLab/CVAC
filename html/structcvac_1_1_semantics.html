string url
 

Detailed Description

Semantics serve to give meaning to label names. The URL, if present, should link to a site that somehow attaches meaning to names. For example, a Label with the name "Europa" could have a link to a Wikipedia page to disambiguate its meaning: http://en.wikipedia.org/wiki/Europa_(moon) Or, as another example, many labels could link to WordNet (http://wordnet.princeton.edu/) to help with explaining their meaning.
