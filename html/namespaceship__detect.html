Detailed Description

Easy!  mini tutorial
Apply a pre-trained ship detector to an image.  The trained model
is stored on the vision.nps.edu server and does not need to be sent
from the client.  The model is customized for ships as seen from
horizontal views, such as from shore or from another ship.
matz 10/11/2013
