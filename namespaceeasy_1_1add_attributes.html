Detailed Description

A very program to add attributes to labelme annoations

Usage:
addAttributes LabelMeImagesFolder LabelMeAnnoationFolder, subfolder,
          attributeName [attributeValue]
Adds that attributeName and possible value to all images in the subfolder.
It adds these in addition to already existing attributes.
